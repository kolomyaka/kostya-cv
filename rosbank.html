UX Тесты

«РОСБАНК» - универсальный финансовый институт с хорошо развитой филиальной сетью, один из крупнейших банков России. Ключевые направления деятельности – розничный бизнес, обслуживание корпоративных клиентов, инвестиционно-банковские услуги, а также private banking.

Продукты, над которыми работал:

  • Раздел приложения “Мои продукты”
  • WEB Desktop Online Bank
  • Дизайн-система

Проблемы

  • Отсутствие виртуальной дебетовой и кредитной карты
  • Отсутствие Интернет банка
  • Отсутсвие компоненов для Desktop версии банка

Решения

  • С нуля разработал удобную механику получения виртуальной карты
  • Участвовал в разработке Интернет банка, полностью подготовил макеты для флоу “Мои продукты” и “Кредитная карта”
  • Создал, описал и удобно расположил новые компоненты специально для Desktop версии банка внутри имеющейся дизайн системы

Раздел “Мои продукты”

  • Согласно исследованиям большое количество пользователей, в силу разных причин, предпочитают использование банковских карт в цифровом виде.  Разработал и интегрировал новый функционал в банковскую систему
